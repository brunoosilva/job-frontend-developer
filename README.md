## Dependencies

- [NodeJs and NPM ()](https://nodejs.org/en/download/)
- Yarn (Optional): Install Yarn use the command: 
    - ```npm i -g yarn```
    
    
I'm going to use Yarn, but you can use NPM too.

## Install

```
yarn
```

OR 

```
npm install
```

## Run Test

```
yarn test
```

OR 

```
npm run test
```

## Run Dev

```
yarn dev
```
OR 

```
npm run dev
```

## Build for Production

```
yarn build
```

OR 

```
npm run test
```
